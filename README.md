# PixelBead Grid Helper

像素珠网格辅助工具，帮助你在珠绣图案上快速添加、编辑和导出网格线。

## 功能特性

- 🎨 **多种主题**：支持暗色标准、亮色标准、暗色像素风、亮色像素风四种界面风格
- 📏 **网格线管理**：
  - 手动添加横线和纵线
  - 拖动调整线位置
  - 批量补全网格线（等间距）
  - 裁剪工具截断网格线
  - 删除选中的线
- ⏪ **撤销/重做**：支持 Ctrl+Z / Ctrl+Y 快捷键
- 📷 **导出功能**：将图片+网格线导出为 PNG 或 JPG 格式

## 安装说明

1. 克隆仓库
2. 创建虚拟环境并激活：
   ```bash
   python -m venv .venv
   .venv\Scripts\Activate.ps1  # Windows
   # 或 source .venv/bin/activate  # Linux/Mac
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用步骤

1. **打开图片**：点击工具栏的"打开"按钮，选择要添加网格线的图片
2. **添加线**：
   - 点击"添加横线"或"添加纵线"
   - 在图片上点击位置添加线
3. **编辑线**：
   - 选中线后拖动可以改变位置
   - 选中线后按 Delete 键或点击"删除选中"按钮删除
4. **批量补全**：
   - 选中一条线
   - 设置间距
   - 点击"补全横线"或"补全纵线"自动等间距补全
5. **裁剪线**：
   - 点击"裁剪"按钮
   - 拖动鼠标画选框
   - 松开鼠标后，相交的线会被截断
   - 按 Esc 键可取消裁剪模式
6. **导出图片**：
   - 点击"导出"按钮
   - 选择保存位置和格式（PNG/JPG）
7. **切换主题**：在工具栏的主题下拉框选择喜欢的风格

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z | 撤销操作 |
| Ctrl+Y | 重做操作 |
| Delete | 删除选中的线 |
| Esc | 取消裁剪模式 |
