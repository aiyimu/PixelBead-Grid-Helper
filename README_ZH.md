## 项目介绍

PixelBead Grid Helper 是一个用于像素珠（拼豆）图案设计的辅助工具。它提供了直观的界面，让你能够在图片上快速添加、编辑和管理网格线，帮助你精确地定位和规划拼豆图案。

## 功能特性

### 🎨 界面主题
- **四种主题风格**：支持暗色标准、亮色标准、暗色像素风、亮色像素风
- 实时主题切换，满足不同使用场景

### 🔍 视图控制
- **缩放功能**：
  - 放大/缩小按钮
  - 100% 显示（原始大小）
  - 适应屏幕（自动调整到窗口大小）
  - 鼠标滚轮缩放（以鼠标位置为中心）
  - 缩放范围：10% - 500%
- **平移功能**：
  - 空格键 + 鼠标左键拖拽
  - 鼠标中键拖拽
  - 画布超出窗口时自动显示滚动条

### 📏 网格线管理
- **添加线条**：
  - 添加横线和纵线
  - 点击图片任意位置添加
- **编辑线条**：
  - 选中后拖拽调整位置
  - 删除选中的线（按钮或 Delete 键）
- **批量补全**：
  - 等间距自动补全横线/纵线
  - 边界留空原则
  - 自动复制原线的所有属性（颜色、粗细、样式）
- **裁剪功能**：
  - 拖拽画选框
  - 仅删除选框内的部分
  - 保留线的两侧部分

### ✏️ 线条属性
- **粗细调整**：
  - 滑块控制（1-10px）
  - 实时预览
- **颜色自定义**：
  - 内置拾色器
  - 支持 RGB/HEX 输入
  - 批量修改多条线
- **样式切换**：
  - 实线
  - 虚线
  - 点线

### 🎯 选择系统
- **单选**：点击单条线选中
- **多选**：Shift + 点击
- **框选**：使用框选工具拖拽选中多条线
- **全选/取消全选**：Ctrl+A / Ctrl+Shift+A
- **选中状态显示**：
  - 高亮显示（红色）
  - 两端和中间的控制点

### 📏 距离计算
- **计算两条平行线之间的距离**：
  - 点击"计算距离"按钮
  - 依次选择两条平行线
  - 显示距离（像素单位）
  - 自动验证两条线是否平行

### ⏪ 撤销/重做
- 支持 Ctrl+Z / Ctrl+Y 快捷键
- 最多记录 50 步历史操作

### 📷 导出功能
- 导出图片 + 网格线
- 支持 PNG 和 JPG 格式
- 默认保存到项目根目录的 `output` 文件夹
- 自动创建输出文件夹

## 安装说明

### 前置要求

- Python 3.7 或更高版本
- pip 包管理器

### 安装步骤

1. **克隆或下载项目**

   ```bash
   git clone <repository-url>
   cd pixelbead-grid-helper
   ```

2. **创建虚拟环境（推荐）**

   Windows:
   ```powershell
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   ```

   Linux/Mac:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```

3. **安装依赖**

   ```bash
   pip install -r requirements.txt
   ```

4. **运行程序**

   ```bash
   python src/main.py
   ```

## 快速开始

1. **打开图片**：点击工具栏的"打开"按钮，选择要添加网格线的图片
2. **添加网格线**：点击"添加横线"或"添加纵线"，然后在图片上点击位置添加
3. **编辑线条**：选中线条后可以拖拽调整位置，或按 Delete 键删除
4. **调整属性**：使用右侧面板的控件调整线条的粗细、颜色和样式
5. **导出图片**：点击"导出"按钮，将图片+网格线保存到 `output` 文件夹

## 详细使用指南

### 1. 打开图片

- 点击工具栏的"打开"按钮
- 在文件对话框中选择图片文件
- 支持的格式：PNG、JPG、JPEG、BMP、GIF、WEBP
- 图片会自动适应屏幕大小显示

### 2. 添加网格线

#### 添加横线
1. 点击工具栏或右侧面板的"添加横线"按钮
2. 按钮会高亮显示，表示进入添加模式
3. 在图片上点击要添加横线的位置
4. 横线会添加在点击的 y 坐标位置

#### 添加纵线
1. 点击工具栏或右侧面板的"添加纵线"按钮
2. 按钮会高亮显示，表示进入添加模式
3. 在图片上点击要添加纵线的位置
4. 纵线会添加在点击的 x 坐标位置

### 3. 选择和编辑线条

#### 选择线条
- **单选**：直接点击单条线
- **多选**：按住 Shift 键，依次点击多条线
- **框选**：
  1. 点击"框选"按钮
  2. 在图片上拖拽画出选框
  3. 松开鼠标后，选框内的所有线都会被选中
- **全选**：按 Ctrl+A 或点击工具栏的"全选"按钮
- **取消选择**：
  - 点击空白处
  - 按 Ctrl+Shift+A
  - 按 Esc 键

#### 编辑线条
- **移动位置**：选中线条后，按住鼠标左键拖拽
- **删除线条**：
  - 选中线条后按 Delete 键
  - 点击右侧面板的"删除选中"按钮
  - 点击工具栏的"删除"按钮

### 4. 调整线条属性

#### 设置默认属性
- 未选中任何线条时，调整属性控件会修改默认值
- 下一条添加的线会使用这些默认属性

#### 修改选中线条的属性
- 选中一条或多条线后，调整属性控件会实时修改所有选中线条
- 支持批量修改

#### 粗细调整
- 使用右侧面板的"粗细"滑块
- 范围：1-10 像素
- 右侧显示当前数值

#### 颜色调整
1. 点击右侧面板的"颜色"按钮（显示当前颜色预览）
2. 在弹出的拾色器中选择颜色
3. 支持 RGB、HEX、HSV 等颜色模式

#### 样式调整
- 使用右侧面板的"样式"下拉框
- 选项：
  - 实线（默认）
  - 虚线
  - 点线

### 5. 批量补全网格线

1. 选中一条横线或纵线
2. 在右侧面板设置"间距"数值（单位：像素）
3. 点击"补全横线"或"补全纵线"按钮
4. 系统会自动等间距补全所有线
5. 新生成的线会复制原线的所有属性（颜色、粗细、样式）

**补全规则**：
- 以选中线为基准，向上/下（横线）或左/右（纵线）等间距复制
- 边界留空：不超出图片边界
- 跳过已存在的位置

### 6. 裁剪网格线

1. 点击工具栏的"裁剪"按钮
2. 按钮会高亮显示，表示进入裁剪模式
3. 在图片上拖拽画出裁剪框
4. 松开鼠标后，与裁剪框相交的部分会被删除
5. 线的两侧部分会保留
6. 按 Esc 键可取消裁剪模式

### 7. 计算两条平行线之间的距离

1. 点击工具栏的"计算距离"按钮
2. 按钮会高亮显示，表示进入计算模式
3. 点击第一条线（横线或纵线）
4. 点击第二条线（必须与第一条线同方向）
5. 弹出对话框显示：
   - 第一条线的位置
   - 第二条线的位置
   - 两条线之间的距离（像素单位）
6. 点击"确定"关闭对话框

### 8. 视图控制

#### 缩放画布
- **按钮控制**：
  - 放大：点击"放大"按钮，每次放大 1.2 倍
  - 缩小：点击"缩小"按钮，每次缩小 0.8 倍
  - 100%：点击"100%"按钮，恢复原始大小
  - 适应屏幕：点击"适应屏幕"按钮，自动调整到窗口大小
- **鼠标滚轮**：
  - 向上滚动：放大
  - 向下滚动：缩小
  - 缩放中心为鼠标当前位置

#### 平移画布
- **空格键 + 左键**：
  1. 按住空格键
  2. 鼠标变为"手型"
  3. 按住左键拖拽平移
  4. 松开空格键退出平移模式
- **鼠标中键**：直接按住中键拖拽平移

### 9. 撤销和重做

- **撤销**：
  - 按 Ctrl+Z
  - 点击工具栏的"撤销"按钮
  - 最多可撤销 50 步
- **重做**：
  - 按 Ctrl+Y
  - 点击工具栏的"重做"按钮

### 10. 导出图片

1. 点击工具栏的"导出"按钮
2. 在保存对话框中选择：
   - 保存位置（默认在项目根目录的 `output` 文件夹）
   - 文件名
   - 文件格式（PNG 或 JPG）
3. 点击"保存"
4. 图片会保存为原始图片尺寸，包含所有网格线

### 11. 切换主题

- 在工具栏的"主题"下拉框中选择主题
- 选项：
  - 暗色标准（默认）
  - 亮色标准
  - 暗色像素风
  - 亮色像素风
- 主题会立即应用到整个界面

## 技术栈

- **Python 3.x** - 编程语言

## 架构设计

项目采用经典的 **MVC (Model-View-Controller)** 架构：

### Model (model.py)
- `ImageModel`：管理图片和网格线数据
- `GridLine`：网格线数据类
- 维护操作历史记录，支持撤销/重做

### View (view.py)
- `MainWindow`：主窗口
- `CanvasWidget`：自定义画布组件
- `HelpDialog`：帮助对话框
- `DistanceDialog`：距离计算对话框
- 四种主题样式定义

### Controller (controller.py)
- 连接 Model 和 View
- 处理用户交互逻辑
- 实现业务功能
